# Skills

Reusable, user-invocable skills (via `/<skill-name>`). Each skill is a directory
containing a `SKILL.md` with frontmatter (`name`, `description`) and instructions.

## How to add a skill
1. Create `.claude/skills/<skill-name>/SKILL.md`.
2. Add frontmatter: `name` and a `description` that says when to use it.
3. Write the instructions the skill should follow when invoked.
