# References

External docs distilled for agent consumption (llms.txt style). Keep entries
short, high-signal, and link back to the source of truth.

## Why this exists
Agents work better from curated, distilled context than from raw upstream docs.
Each file here is a compact summary an agent can load cheaply, with a link to the
full source when depth is needed.

## How to add a reference
1. Create `<topic>.md`.
2. Start with the source URL and a one-line "what this is".
3. Distill only what an agent needs to act — not a full copy of the source.
